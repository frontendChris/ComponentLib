Page Templates

Center Aligned Template

Center aligned template. It's width is set to 960px by default

<!DOCTYPE html>
<html>
<head>
     <meta charset="utf-8">
     <meta http-equiv="X-UA-Compatible" content="IE=edge">
     <title>Some title</title>
     <meta name="format-detection" content="telephone=no">
     <meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">
</head>
<body class="layout_center">
     <div class="page_container_outer">
         <div class="page_container_inner">
           ...
         </div>
     </div>
</body>

The page you are currently viewing is an example of a center aligned layout

Left Sidebar Template

Layout template that has a fixed left-hand sidebar and a fluid content area.

<html>
<head>
     <meta charset="utf-8">
     <meta http-equiv="X-UA-Compatible" content="IE=edge">
     <title>Some title</title>
     <meta name="format-detection" content="telephone=no">
     <meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">
</head>
<body class="layout_side_left">
     <div class="page_container_outer">
         <div class="page_container_inner">
               <div class="sidebar_left">
                    Sidebar Content
               </div>
               <div class="push_left page_container_inner">
                    Main Content
               </div>
         </div>
     </div>
</body>
